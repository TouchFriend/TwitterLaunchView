# How to use
1.Import header
 ```` objective-c
  #import "NJLaunchView.h"
 ```` 
2.use class method
 ```` objective-c
  - (void)viewDidAppear:(BOOL)animated
  {
      [super viewDidAppear:animated];
      [NJLaunchView addLaunchView];
  }
 ````
